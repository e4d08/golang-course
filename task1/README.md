# Домашнее задание №1  
## CLI-инструмент для получения информации о репозитории GitHub
### Функциональность
Утилита получает и выводит в стандартный поток вывода следующую информацию о репозитории GitHub:
- Название,
- Описание,
- Количество звёзд,
- Количество форков,
- Дата создания,
- Лицензия.
### Сборка
`go build`
### Запуск
Для запуска необходимо передать утилите в качестве единственного аргумента ссылку на любую страницу репозитория:
`gh-anal https://github.com/<owner>/<name>/...`
`gh-anal https://github.com/hyprwm/Hyprland/pull/13636`
